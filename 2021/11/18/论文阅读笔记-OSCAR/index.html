论文阅读笔记-OSCAR

论文要点

  • 将原来的数据形式image-text对转化为三元组形式 (a word sequence, a set of objects tags, a set of image region features) 。这些objects tags可以作为锚点，将image-text中互相对应的部分（word与objects）对齐。

模型结构

模型输入

模型输入为（w,q,v）三元组，分别为word token，objects tags，region features。其中q作为锚点anchor point对齐两个模态。w是将text经由word embedding之后的序列。v，q的产生过程如下：给定一张含有K个objects的图像（一般情况有过采样和噪声），Faster RCNN用于将每个region提取视觉语义为 (v′,z)(v',z)(v′,z)，其中v’是一个P维的向量，如P=2048，z是R维的每个region的位置向量（如R=4或6）。将v‘和z组合concatenate起来形成一个位置敏感的region feature embedding，再用一个线性投影将其转化为与word embedding同维度的向量表示v（适应BERT的输入维度）。同时，这个Faster RCNN还被用来提取一系列的高预测率的objects tags，q是这些objects的word embedding序列。

预训练目标

模型的输入可以从两个视角看：

Dictionary View: Masked Token Loss(x’)

object tags(q)与word tokens(w)共享语言语义空间，image region features位于视觉语义空间。定义一个离散token序列 $h \triangleq
[w,q] $，应用于Masked Token Loss。在每一个iteration，以15%的概率对h的每个输入token随机mask，并将被mask的 hih_ihi​ 替换为特殊的token [MASK]。目标是根据没mask的token和image feature v来预测这些被mask的token，方法为最小化negative log-likelihood 负log最大似然：

LMTL=−E(v,h)∼Dlog⁡p(hi∣h∖i,v)\mathcal L_{MTL}=-\mathbb E_{(v,h)\thicksim \mathcal D} \log p(h_i|h_{\setminus i},v) LMTL​=−E(v,h)∼D​logp(hi​∣h∖i​,v)

Modality View: Contrastive Loss(x)

根据图像模态的信息和语言模态的信息将其分为图像模态：$h’ \triangleq[q,v] $，语言模态：w。接着将q用50%的概率替换为dataset中一个不同的tag，即被污染的负样本。将[CLS]标签输出的token作为（h’, w）的联合表示，在对其应用一个全连接层作为二分类器 f(.)f(.)f(.) 来分辨是为原始样本（y=1）还是被污染的样本（y=2）。这个对比学习的loss被定义为：

LC=−E(h′,w)∼Dlog⁡p(y∣f(h′,w))\mathcal L_{C}=-\mathbb{E}_{(h',w)\thicksim \mathcal D}\log p(y|f(h',w)) LC​=−E(h′,w)∼D​logp(y∣f(h′,w))

整个预训练Loss为：

LPre−training=LMTL+LC\mathcal L_{Pre-training}=\mathcal L_{MTL}+\mathcal L_{C} LPre−training​=LMTL​+LC​

预训练数据集

COCO，Conceptual Captions (CC) ，SBU captions，ﬂicker30k，GQA 共4.1million数据，包含6.5million三元组。

模型配置

模型有两种配置，OSCARB和OSCARLOSCAR_B和OSCAR_LOSCARB​和OSCARL​ Multi layer transformer部分用BERT初始化即分别BERT-base（H=768）和BERT-large（H=1024）
