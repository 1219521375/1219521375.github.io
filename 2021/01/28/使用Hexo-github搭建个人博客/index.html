使用Hexo&github搭建个人博客

一、环境与准备工作

  • github账号

  • 安装nodejs npm git

本文环境

  • win10

  • node.js v10.13

二、搭建gitpage博客

搭建仓库

新建一个名为你的用户名.github.io的仓库，同时也是你的博客访问地址，比如http://yourname.github.io。 此仓库默认开启gitpages功能。

绑定域名

以阿里云为例，购买一个域名之后在域名解析中配置一个CNAME记录，使得你的域名可以定向到yourname.github.io。

添加CNAME文件

在github.io项目根目录创建一个名为CNAME的文件（无后缀），填写域名，将你的域名与github.io绑定

绑定ssh key

链接本地与github，具体见百度。
