CS231n-assignment1

CS231n的作业一主要有以下内容：KNN、SVM、softmax、Two Layer Neural NetWork，本文主要记录算法思想，代码上传至https://github.com/1219521375/CS231n2020。

KNN

knn是一个非常简单的机器学习算法，其思想可以用一句话概括，即输出与输入样本最相似的k个样本的标签。那么算法的核心就在于如何界定两个样本的相似度上，常用的有L1距离（即曼哈顿距离），L2距离，余弦距离等。作业里使用的是L2距离：

distance(x,y)=∑i=0N(xi−yi)2distance(x,y) = \sqrt{\sum_{i=0}^N(x_i-y_i)^2} distance(x,y)=i=0∑N​(xi​−yi​)2​

在图像间距离的计算上，简单粗暴地将32x32x3的矩阵拉成一个3072维的向量，计算每个像素点的差值。

具体任务上，KNN算法没有传统机器学习任务的先训练后预测的步骤，所有的数据样本就是它的模型。当一个新样本开始预测时，计算它到所有当前数据样本的距离；当有x个待预测样本，y个数据样本时，就需要计算一个[x,y]的距离矩阵。

Two Layer Net

两层全连接神经网络。重要点是理解前向传播与反向传播。

两层全连接神经网络的结构如下：

input - fully connected layer - ReLU - fully connected layer - softmax

其中relu是一个简单的max(0,x)max(0,x)max(0,x)函数，softmax函数在上节中讲到，将每一类的最终得分转化成相加和为1的概率。

而剩下的就是一个全连接层(fully connected layer)了。单层的全连接层其实就是一个线性回归，公式如下：$$ y=W*x+b $$。假设输入为N个D维向量，即x=[N,D]x=[N,D]x=[N,D]，那么权重矩阵即为W=[D,H]W=[D,H]W=[D,H]，HHH是所谓全连接层神经元的数量，就是神经网络的宽度。至此，神经网络的前向传播已经显而易见，将输入一层层经过上述网络层即可。

反向传播

反向传播是利用链式法则递归计算表达式的梯度的方法。理解反向传播过程及其精妙之处，对于理解、实现、设计和调试神经网络非常关键。

神经网络使用梯度来更新权重，梯度即导数是函数上升最快的方向，那么梯度的负向即减去梯度就可以使函数不断下降至最低点，就可以拟合函数。

先从简单表达式入手
  • 考虑一个简单二元函数f(x,y)=xyf(x,y) = xyf(x,y)=xy，对两个变量分别求偏导，dfdx=y,dfdy=x\frac{df}{dx}=y,\frac{df}{dy}=xdxdf​=y,dydf​=x，也就是说，对于二元乘法来说，变量的梯度（导数）为它所乘的另一个变量。
  • 对于简单的加法函数f(x,y)=x+yf(x,y)=x+yf(x,y)=x+y，容易看出x,yx,yx,y的梯度（导数）均为1。
  • 对于最大值函数（relu）f(x,y)=max(x,y)f(x,y)=max(x,y)f(x,y)=max(x,y)，求导得dfdx=1(x≥y),dfdy=1(y≥x)\frac{df}{dx}=1(x\ge y),\frac{df}{dy}=1(y\ge x)dxdf​=1(x≥y),dydf​=1(y≥x)，可以看出，如果一个变量比另一个变量大，那么梯度是1，反之为0。
使用链式法则计算复合表达式
  • 考虑一个复合函数f(x,y,z)=(x+y)∗zf(x,y,z) = (x+y)*zf(x,y,z)=(x+y)∗z，使用链式法则将表达式拆分成q=x+yq=x+yq=x+y和f=qzf=qzf=qz，容易看出∂f∂q=z,∂f∂z=q\frac{\partial f}{\partial q}=z,\frac{\partial f}{\partial z}=q∂q∂f​=z,∂z∂f​=q，而qqq对x,yx,yx,y的偏导都为1，我们需要求fff对于x,y,zx,y,zx,y,z的梯度，链式法则指出将这些梯度表达式链接起来的正确方式是相乘，如∂f∂x=∂f∂q∗∂q∂x\frac{\partial f}{\partial x}=\frac{\partial f}{\partial q}*\frac{\partial q}{\partial x}∂x∂f​=∂q∂f​∗∂x∂q​，实际操作中算出两个简单梯度后相乘即可。

上述操作可以用上面的计算图表示，绿色数字为前向传播(−2+5)∗−4=−12(-2+5)*-4=-12(−2+5)∗−4=−12，红色数字为反向传播，假设fff处得到的回传梯度为1，经过一个乘法门，梯度为相乘的变量的值，所以qqq的梯度为1∗z=−41*z=-41∗z=−4，zzz的梯度为1∗q=31*q=31∗q=3。而加法门的梯度均为1，所以x,yx,yx,y的梯度均为-4。

反向传播小结

  • 加法门将输出的梯度相等地分发给它所有的输入，这一行为与输入值在前向传播时的值无关。这是因为加法操作的局部梯度都是简单的+1，所以所有输入的梯度实际上就等于输出的梯度，因为乘以1.0保持不变。
  • 取最大值门单元对梯度做路由。和加法门不同，取最大值门将梯度转给其中一个输入，这个输入是在前向传播中值最大的那个输入。这是因为在取最大值门中，最高值的局部梯度是1.0，其余的是0。
  • 乘法门单元相对不容易解释。它的局部梯度就是输入值，但是是相互交换之后的，然后根据链式法则乘以输出值的梯度。

注意一种比较特殊的情况，如果乘法门单元的其中一个输入非常小，而另一个输入非常大，那么乘法门的操作将会不是那么直观：它将会把大的梯度分配给小的输入，把小的梯度分配给大的输入。在线性分类器W∗xW*xW∗x中，这表示输入数据的大小对于权重梯度的大小有影响，这就是为什么数据预处理关系重大，它即使只是有微小变化，也会产生巨大影响。

以上所有概念都适用于矩阵和向量操作，只需要注意维度和转置操作。
